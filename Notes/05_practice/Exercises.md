![[Pasted image 20260326224324.png]]
* Host name for PC is: **webserv** and domain name is: **non-real.com**.
* Here **host A** want to reach some website on the internet.
* **Default gateway must be the router interface's IP** .
* The **default gateway for the router itself must be the ISP** i.e Internet service provider (like zain, umniah, orange,...).
* The internet itself needs to send packets to our host i.e PC, so the **destination is: PC IP**, to reach it, it's must send to the **Next hop**: router interface's IP.
