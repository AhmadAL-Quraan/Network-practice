# Network-practice
This is a general practice of computer networking.

# Topic covered 

* OSI layers
* TCP/IP layers
* Subnet mask
* Default gateway 
* LAN and WAN 



# Run

* Run the code practice throw localhost using any http server ex: `python -m http.server 42`.
